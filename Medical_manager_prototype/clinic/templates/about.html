About Page

This medical portal was developed by DIS-project group 66, 2024:

  • Nynne West Kristensen, bkf248
  • Benjamin Daniel Nyberg Shultz, mvf630

You can register new patient accounts through the portal. Doctor accounts must be entered by making changes to the DML-script schema_ins.sql. To log in, we have created some test accounts. The patient login can be accessed using a patient CPR number in the range 1001 to 1005 with the password '1'. The doctor login can be accessed using doctor ID 1 to 5 and the same password.

To add more sample data, make your changes to the DML-script schema_ins.sql.

This is a prototype. We have implemented several key functions, including patient registration, appointment scheduling, medical record management, prescription tracking, and more. Further enhancements and additional features can be built on top of this foundation.

The project is hosted on GitHub at this link.
